## 🧑‍💻 About Me

I'm **Anni** — a 3rd year B.Tech student from India who builds things that matter. Not just academic projects, but tools with real users and real impact. I'm equally comfortable writing backend logic in Java, building REST APIs with Spring Boot, automating tasks in Python, or wiring up hardware with Arduino.

Outside of code, I run **Vidya Sampadana** — an education nonprofit I co-founded that helps students prepare for KCET & NEET, get scholarships, and access free books and mentors.
